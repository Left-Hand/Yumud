# SR2631Z3 NMEA0183 协议报文详解（GGA/GLL/GSA/GSV/RMC/VTG/ZDA/TXT）
本文档基于SR2631Z3规格书第7–14页整理，完整复现各NMEA报文格式、字段、示例与说明。

---

## 3.1 GGA（时间、位置、定位类型）
**描述**：输出接收机UTC时间、经纬度、定位质量、卫星数、水平精度、海拔高度等核心定位数据。
**类型**：输出
**格式**：`$--GGA,UTCtime,lat,uLat,lon,uLon,FS,numSv,HDOP,msl,uMsl,sep,uSep,diffAge,diffSta*CS<CR><LF>`
**示例**：`$BDGGA,023656.00,2240.61563,N,11359.86512,E,1,23,0.7,96.53,M,-3.52,M,,*5B`

| 字段 | 名称 | 格式 | 参数说明 |
| ---- | ---- | ---- | ---- |
| 1 | $--GGA | 字符串 | 消息ID，GGA语句头，`--`为系统标识 |
| 2 | UTCtime | hhmmss.ss | UTC时间 |
| 3 | lat | ddmm.mmmmm | 纬度，前2位为度，后续为分 |
| 4 | uLat | 字符 | 纬度方向：N=北，S=南 |
| 5 | lon | dddmm.mmmm | 经度，前3位为度，后续为分 |
| 6 | uLon | 字符 | 经度方向：E=东，W=西 |
| 7 | FS | 数值 | 定位质量标志（不可为空） |
| 8 | numSv | 数值 | 参与定位的卫星数量 |
| 9 | HDOP | 数值 | 水平精度因子 |
| 10 | msl | 数值 | 海拔高度（相对大地水准面） |
| 11 | uMsl | 字符 | 高度单位，固定为M（米） |
| 12 | sep | 数值 | 高程异常（大地高-海拔高） |
| 13 | uSep | 字符 | 高程异常单位，固定为M（米） |
| 14 | diffAge | 数值 | 差分数据龄期，未用DGPS则为空 |
| 15 | diffSta | 数值 | 差分参考站ID |
| 16 | CS | 16进制 | 校验和（$与*之间字符异或） |
| 17 | <CR><LF> | 字符 | 回车换行 |

**定位质量标志FS**
| 数值 | 描述 |
| ---- | ---- |
| 0 | 定位不可用/无效 |
| 1 | SPS单点定位，有效 |
| 2 | 差分SPS，有效 |
| 3 | PPS精密定位，有效 |
| 4 | RTK固定解 |
| 5 | RTK浮动解 |
| 6 | 估算/航位推算 |

## 3.2 GLL（经纬度、定位时间、状态）
**描述**：输出纬度、经度、UTC时间、数据有效性与定位模式。
**类型**：输出
**格式**：`$--GLL,lat,uLat,lon,uLon,UTCtime,valid,mode*CS<CR><LF>`
**示例**：`$BDGLL,2240.61563,N,11359.86512,E,023656.00,A,A*78`

| 字段 | 名称 | 格式 | 参数说明 |
| ---- | ---- | ---- | ---- |
| 1 | $--GLL | 字符串 | 消息ID，GLL语句头 |
| 2 | lat | ddmm.mmmmm | 纬度 |
| 3 | uLat | 字符 | N/S |
| 4 | lon | dddmm.mmmm | 经度 |
| 5 | uLon | 字符 | E/W |
| 6 | UTCtime | hhmmss.ss | 定位UTC时间 |
| 7 | valid | 字符 | 数据有效性 |
| 8 | mode | 字符 | 定位模式 |
| 9 | CS | 16进制 | 校验和 |
| 10 | <CR><LF> | 字符 | 回车换行 |

**数据有效性valid**
| 字符 | 描述 |
| ---- | ---- |
| A | 数据有效 |
| V | 数据无效 |

**定位模式mode**
| 字符 | 描述 |
| ---- | ---- |
| A | 自主定位 |
| E | 估算/航位推算 |
| N | 数据无效 |
| D | 差分模式 |
| M | 未定位但有外部/历史位置 |

## 3.3 GSA（卫星状态、DOP值）
**描述**：输出参与定位的卫星编号、定位模式、PDOP/HDOP/VDOP精度因子。多系统时每个系统输出1–2条。
**类型**：输出
**格式**：`$--GSA,smode,FS{,SVID},PDOP,HDOP,VDOP*CS<CR><LF>`
**示例**：
`$BDGSA,A,3,01,02,03,04,05,06,07,09,10,16,19,20,1.0,0.7,0.8,4*30`
`$BDGSA,A,3,22,27,28,30,36,37,39,40,46,59,60,,1.0,0.7,0.8,4*3E`

| 字段 | 名称 | 格式 | 参数说明 |
| ---- | ---- | ---- | ---- |
| 1 | $--GSA | 字符串 | 消息ID，GSA语句头 |
| 2 | smode | 字符 | 模式切换指示 |
| 3 | FS | 数字 | 定位状态标志 |
| 4 | {,SVID} | 数值 | 定位卫星编号（最多12颗，不足补空） |
| 5 | PDOP | 数值 | 位置精度因子 |
| 6 | HDOP | 数值 | 水平精度因子 |
| 7 | VDOP | 数值 | 垂直精度因子 |
| 8 | systemId | 数值 | NMEA4.11+系统ID |
| 9 | CS | 16进制 | 校验和 |
| 10 | <CR><LF> | 字符 | 回车换行 |

**模式切换smode**
| 字符 | 描述 |
| ---- | ---- |
| M | 手动切换（2D/3D强制） |
| A | 自动切换（2D/3D） |

**定位状态FS**
| 数值 | 描述 |
| ---- | ---- |
| 1 | 定位无效 |
| 2 | 2D定位 |
| 3 | 3D定位 |

## 3.4 GSV（可见卫星信息）
**描述**：输出可见卫星编号、仰角、方位角、载噪比。每条最多4颗卫星，多卫星分多条输出。
**类型**：输出
**格式**：`$--GSV,numMsg,msgNo,numSv{,SVID,ele,az,cn0}*CS<CR><LF>`
**示例**：`$BDGSV,6,1,23,01,45,125,38,02,46,235,40,03,61,189,46,04,32,112,37,1*7B`

| 字段 | 名称 | 格式 | 参数说明 |
| ---- | ---- | ---- | ---- |
| 1 | $--GSV | 字符串 | 消息ID，GSV语句头 |
| 2 | numMsg | 字符 | 本批次总语句数 |
| 3 | msgNo | 数字 | 当前语句编号 |
| 4 | numSv | 数值 | 可见卫星总数 |
| 5 | {,SVID,ele,az,cn0} | 数值 | 卫星编号/仰角(0~90°)/方位角(0~359°)/载噪比(0~99dBHz) |
| 6 | signalId | 数值 | GNSS信号ID |
| 7 | CS | 16进制 | 校验和 |
| 8 | <CR><LF> | 字符 | 回车换行 |

## 3.5 RMC（推荐最小定位信息）
**描述**：最常用精简定位帧，含时间、日期、经纬度、速度、航向、定位模式。
**类型**：输出
**格式**：`$--RMC,UTCtime,status,lat,uLat,lon,uLon,spd,cog,date,mv,mvE,mode*CS<CR><LF>`
**示例**：`$BDRMC,023656.00,A,2240.61563,N,11359.86512,E,0.16,,140324,,,A,V*2C`

| 字段 | 名称 | 格式 | 参数说明 |
| ---- | ---- | ---- | ---- |
| 1 | $--RMC | 字符串 | 消息ID，RMC语句头 |
| 2 | UTCtime | hhmmss.ss | UTC时间 |
| 3 | status | 字符 | A=有效，V=无效 |
| 4 | lat | ddmm.mmmmm | 纬度 |
| 5 | uLat | 字符 | N/S |
| 6 | lon | dddmm.mmmm | 经度 |
| 7 | uLon | 字符 | E/W |
| 8 | spd | 数值 | 对地速度（节） |
| 9 | cog | 数值 | 对地真航向（度） |
| 10 | date | ddmmyy | 日期 |
| 11 | mv | 数值 | 磁偏角（固定为空） |
| 12 | mvE | 字符 | 磁偏角方向（固定为空） |
| 13 | mode | 字符 | 定位模式 |
| 14 | navStatus | 字符 | 导航状态（V=不输出） |
| 15 | CS | 16进制 | 校验和 |
| 16 | <CR><LF> | 字符 | 回车换行 |

**定位模式mode**（同GLL）
| 字符 | 描述 |
| ---- | ---- |
| A | 自主 |
| E | 估算 |
| N | 无效 |
| D | 差分 |
| M | 未定位有历史位置 |

## 3.6 VTG（地面速度、航向）
**描述**：输出对地真航向、磁航向、速度（节/公里/小时）。
**类型**：输出
**格式**：`$--VTG,cogt,T,cogm,M,sog,N,kph,K,mode*CS<CR><LF>`
**示例**：`$BDVTG,,,,,0.16,N,0.30,K,A*2F`

| 字段 | 名称 | 格式 | 参数说明 |
| ---- | ---- | ---- | ---- |
| 1 | $--VTG | 字符串 | 消息ID，VTG语句头 |
| 2 | cogt | 数值 | 真北航向（度） |
| 3 | T | 字符 | 真北指示，固定T |
| 4 | cogm | 数值 | 磁北航向（度） |
| 5 | M | 字符 | 磁北指示，固定M |
| 6 | sog | 数值 | 速度（节） |
| 7 | N | 字符 | 节单位，固定N |
| 8 | kph | 数值 | 速度（km/h） |
| 9 | K | 字符 | km/h单位，固定K |
| 10 | mode | 字符 | 定位模式 |
| 11 | CS | 16进制 | 校验和 |
| 12 | <CR><LF> | 字符 | 回车换行 |

**定位模式mode**（同GLL/RMC）

## 3.7 ZDA（时间、日期）
**描述**：纯时间日期输出，不含定位信息。
**类型**：输出
**格式**：`$--ZDA,UTCtime,day,month,year,ltzh,ltzn*CS<CR><LF>`
**示例**：`$BDZDA,023656.00,14,03,2024,00,00*71`

| 字段 | 名称 | 格式 | 参数说明 |
| ---- | ---- | ---- | ---- |
| 1 | $--ZDA | 字符串 | 消息ID，ZDA语句头 |
| 2 | UTCtime | hhmmss.ss | UTC时间 |
| 3 | day | 数值 | 日（01–31） |
| 4 | month | 数值 | 月（01–12） |
| 5 | year | 数值 | 年（4位） |
| 6 | ltzh | 数值 | 时区小时，固定00 |
| 7 | ltzn | 数值 | 时区分钟，固定00 |
| 8 | CS | 16进制 | 校验和 |
| 9 | <CR><LF> | 字符 | 回车换行 |

## 3.8 TXT（天线状态检测）
**描述**：模块输出天线状态文本，用于诊断天线开路/短路/正常。
**类型**：输出
**格式**：`$GPTXT,01,01,01,string*hh<CR><LF>`
**示例**：`$GPTXT,01,01,01,ANTENNA OPEN*25`

| 字段 | 名称 | 格式 | 参数说明 |
| ---- | ---- | ---- | ---- |
| 1 | $GPTXT | 字符串 | 消息ID，TXT语句头 |
| 2 | xx | 数值 | 总语句数，固定01 |
| 3 | yy | 数值 | 语句编号，固定01 |
| 4 | zz | 数值 | 文本标识符，固定01 |
| 5 | string | 字符串 | 天线状态文本 |
| 6 | CS | 16进制 | 校验和 |
| 7 | <CR><LF> | 字符 | 回车换行 |

**天线状态string**
| 文本 | 含义 |
| ---- | ---- |
| ANTENNA OPEN | 天线开路 |
| ANTENNA OK | 天线正常 |
| ANTENNA SHORT | 天线短路 |
