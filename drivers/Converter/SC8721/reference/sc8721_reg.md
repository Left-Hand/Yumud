## 10.1 0x01H CSO 寄存器
| 位   | 读写模式 | 位名称 | 上电复位默认值 | 描述 | 备注 |
| ---- | -------- | ------ | -------------- | ---- | ---- |
| 5    | R/W      | CSO_SET | 1              | 文档未补充具体功能描述 | -    |
| 7-6、4-0 | R/W | 保留 | 1/0 | 内部使用，禁止改写 | - |

## 10.2 0x02H SLOPE_COMP 斜率补偿寄存器
| 位   | 读写模式 | 位名称 | 上电复位默认值 | 描述 | 备注 |
| ---- | -------- | ------ | -------------- | ---- | ---- |
| 7-2  | R/W      | 保留   | 000000         | 内部使用，禁止改写 | -    |
| 1-0  | R/W      | SLOPE_COMP | 00      | 用于设置输出电压线压降补偿：<br>00：无斜率补偿（默认）<br>01：50mV/A<br>10：100mV/A<br>11：150mV/A | - |

## 10.3 0x03H VOUT_SET_MSB 输出电压设置高位寄存器
| 位   | 读写模式 | 位名称 | 上电复位默认值 | 描述 | 备注 |
| ---- | -------- | ------ | -------------- | ---- | ---- |
| 7-0  | R/W      | VOUT_SET_MSB | 00000000    | 输出电压设置值的高8位，与VOUT_SET_LSB配合设置输出电压<br>计算公式：<br>VOUT_SET = (VOUT_SET_MSB × 4 + VOUT_SET_LSB) × 20mV | - |

## 10.4 0x04H VOUT_SET_LSB 输出电压设置低位寄存器
| 位   | 读写模式 | 位名称 | 上电复位默认值 | 描述 | 备注 |
| ---- | -------- | ------ | -------------- | ---- | ---- |
| 7-5  | R/W      | 保留   | 000            | 内部使用，禁止改写 | -    |
| 4    | R/W      | FB_SEL | 0              | 仅放电模式下的输出电压设置控制：<br>0：外部输出电压设置，由FB引脚电阻分压设置输出电压<br>1：内部输出电压设置，由VOUT_SET_MSB、VOUT_SET_LSB和FB_DIR共同设置 | - |
| 3    | R/W      | FB_ON  | 0              | 输出电压设置屏蔽位：<br>0：禁止修改输出电压<br>1：允许修改输出电压 | - |
| 2    | R/W      | FB_DIR | 0              | 输出电压设置方向：<br>0：递增（默认）<br>1：递减 | - |
| 1-0  | R/W      | VOUT_SET_LSB | 00        | 输出电压设置值的低2位 | - |

## 10.5 0x05H GLOBAL_CTRL 全局控制寄存器
| 位   | 读写模式 | 位名称 | 上电复位默认值 | 描述 | 备注 |
| ---- | -------- | ------ | -------------- | ---- | ---- |
| 7-3  | R/W      | 保留   | 00000          | 内部使用，禁止改写 | -    |
| 2    | R/W      | DIS_DCDC | 0            | DC-DC开关控制：<br>0：使能（默认）<br>1：禁用 | - |
| 1    | R/W      | REG_LOAD | 0            | 写1加载寄存器（CSO_SET/VOUT_SET/FB_DIR），读取后自动清0 | - |
| 0    | R/W      | 保留   | 0              | 内部使用，禁止改写 | -    |

## 10.6 0x06H SYS_SET 系统设置寄存器
| 位   | 读写模式 | 位名称 | 上电复位默认值 | 描述 | 备注 |
| ---- | -------- | ------ | -------------- | ---- | ---- |
| 7    | R/W      | EN_PWM | 1              | PWM模式使能：<br>0：禁用，自动PWM模式<br>1：使能，强制PWM模式（默认） | - |
| 6    | R/W      | EXT_DT | 0              | 开关死区时间设置：<br>0：20ns（默认）<br>1：40ns | - |
| 5    | R/W      | 保留   | 0              | 内部使用，禁止改写 | -    |
| 4    | R/W      | EN_VINREG | 1            | 输入电压稳压使能控制：<br>0：禁用<br>1：使能（默认） | - |
| 3-0  | R/W      | 保留   | 0000           | 内部使用，禁止改写 | -    |

## 10.8 0x08H FREQ_SET 频率设置寄存器
| 位   | 读写模式 | 位名称 | 上电复位默认值 | 描述 | 备注 |

## 10.9 0x09H STATUS_1 状态寄存器1

| 0    | R        | OCP    | 0              | 过流保护标志位：1表示触发内部过流保护 | - |
