# SC8815 寄存器手册
## 基础通信信息
- 7位从机地址：`0x74`
- 8位写命令地址：`0xE8`
- 8位读命令地址：`0xE9`

## 2 寄存器详细描述
### 2.1 0x00H VBAT_SET 电池电压设置寄存器 (R/W)
| 位     | 位名称      | 上电复位默认值 | 描述                                                                 | 备注                  |
| ------ | ----------- | -------------- | -------------------------------------------------------------------- | --------------------- |
| 7-6    | IRCOMP      | 00             | 电池内阻补偿设置：00=0mΩ(默认)；01=20mΩ；10=40mΩ；11=80mΩ            | PSTOP引脚高电平时配置 |
| 5      | VBAT_SEL    | 0              | 电池电压设置选择：0=内部设置(默认)；1=外部设置                       | PSTOP引脚高电平时配置 |
| 4-3    | CSEL        | 00             | 电池串数选择（仅内部设置有效）：00=1串(默认)；01=2串；10=3串；11=4串 | PSTOP引脚高电平时配置 |
| 2-0    | VCELL_SET   | 001            | 单节电池电压设置（仅内部设置有效）：<br>000=4.1V；001=4.2V(默认)；<br>010=4.25V；011=4.3V；<br>100=4.35V；101=4.4V；<br>110=4.45V；111=4.5V | PSTOP引脚高电平时配置 |

### 2.2 0x01H VBUSREF_I_SET 内部VBUS参考电压设置寄存器 (R/W)
| 位     | 位名称          | 上电复位默认值 | 描述                                                                 | 备注 |
| ------ | --------------- | -------------- | -------------------------------------------------------------------- | ---- |
| 7-0    | VBUSREF_I_SET   | 0011 0001(49)  | 内部VBUS参考电压高8位设置，取值范围0~255，默认值49                   | -    |
**计算公式**：
- 内部参考电压：`VBUSREF_I = (4×VBUSREF_I_SET + 3 + 1)×2mV`
- 默认参考电压：`(4×49+4)×2mV = 400mV`
- FB_SEL=0时默认VBUS输出电压：`400mV × 12.5 = 5V`

### 2.3 0x02H VBUSREF_I_SET2 内部VBUS参考电压低2位寄存器 (R/W)
| 位     | 位名称            | 上电复位默认值 | 描述                   | 备注 |
| ------ | ----------------- | -------------- | ---------------------- | ---- |
| 7-6    | VBUSREF_I_SET2    | 11             | 内部VBUS参考电压低2位设置 | -    |
| 5-0    | 保留              | xx xxxx        | 内部使用，禁止改写     | -    |

### 2.4 0x03H VBUSREF_E_SET 外部VBUS参考电压设置寄存器 (R/W)
| 位     | 位名称          | 上电复位默认值 | 描述                                                                 | 备注 |
| ------ | --------------- | -------------- | -------------------------------------------------------------------- | ---- |
| 7-0    | VBUSREF_E_SET   | 0111 1100(124) | 外部VBUS参考电压高8位设置，取值范围0~255，默认值124                  | -    |
**默认参考电压**：`(4×124+4)×2mV = 1V`

### 2.5 0x04H VBUSREF_E_SET2 外部VBUS参考电压低2位寄存器 (R/W)
| 位     | 位名称            | 上电复位默认值 | 描述                                                                 | 备注 |
| ------ | ----------------- | -------------- | -------------------------------------------------------------------- | ---- |
| 7-6    | VBUSREF_E_SET2    | 11             | 外部VBUS参考电压低2位设置，取值0~3：00=0；01=1；10=2；11=3(默认)     | -    |
| 5-0    | 保留              | xx xxxx        | 内部使用，禁止改写                                                   | -    |
**计算公式**：
- 外部参考电压：`VBUSREF_E = (4×VBUSREF_E_SET + VBUSREF_E_SET2 + 1)×2mV`
- VBUS输出电压：`VBUS = VBUSREF_E × (1 + R_UP/R_DOWN)`（R_UP/R_DOWN为FB引脚分压电阻）

### 2.6 0x05H IBUS_LIM_SET 总线电流限流设置寄存器 (R/W)
| 位     | 位名称        | 上电复位默认值 | 描述                                                                 | 备注 |
| ------ | ------------- | -------------- | -------------------------------------------------------------------- | ---- |
| 7-0    | IBUS_LIM_SET  | 1111 1111(255) | 总线电流限流值设置，取值范围0~255，充电/放电模式均有效，限流值≥300mA  | -    |
**计算公式**：`IBUS_LIM(A) = (IBUS_LIM_SET + 1)/256 × IBUS_RATIO × 10mΩ/R_S1`
- R_S1：VBUS侧电流采样电阻
- 示例（R_S1=10mΩ）：`(255+1)/256 ×3 ×10mΩ/10mΩ = 3A`（默认限流3A）

### 2.7 0x06H IBAT_LIM_SET 电池电流限流设置寄存器 (R/W)
| 位     | 位名称        | 上电复位默认值 | 描述                                                                 | 备注 |
| ------ | ------------- | -------------- | -------------------------------------------------------------------- | ---- |
| 7-0    | IBAT_LIM_SET  | 1111 1111(255) | 电池电流限流值设置，取值范围0~255，限流值≥300mA                      | -    |
**计算公式**：`IBAT_LIM(A) = (IBAT_LIM_SET + 1)/256 × IBAT_RATIO × 10mΩ/R_S2`
- R_S2：BAT侧电流采样电阻
