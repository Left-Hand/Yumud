## 8.5.1 系统状态寄存器(SYS_STAT - 0x00)
| 位 | 名称 | 访问 | 描述 | 复位值 |
| ---- | ---- | ---- | ---- | ------ |
| 7 | CC_READY | RW | 库仑计新读数就绪标志：0=无新读数/已清零；1=有新读数（主机写1清零） | 0 |
| 6 | RSVD | RW | 保留位，禁止使用 | 0 |
| 5 | DEVICE_XREADY | RW | 芯片内部故障标志：0=芯片正常；1=检测到内部故障（主机写1清零） | 0 |
| 4 | OVRD_ALERT | RW | ALERT引脚外部上拉标志：0=无外部覆写；1=检测到外部覆写（主机写1清零） | 0 |
| 3 | UV | RW | 欠压故障标志：0=无欠压；1=检测到欠压（主机写1清零） | 0 |
| 2 | OV | RW | 过压故障标志：0=无过压；1=检测到过压（主机写1清零） | 0 |
| 1 | SCD | RW | 放电短路故障标志：0=无短路；1=检测到短路（主机写1清零） | 0 |
| 0 | OCD | RW | 放电过流故障标志：0=无过流；1=检测到过流（主机写1清零） | 0 |
**注**：写0不改变对应位状态，写1清除对应位状态。
