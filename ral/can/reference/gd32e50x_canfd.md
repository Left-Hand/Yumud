# GD32E50x CAN/FD 寄存器字段说明
## 26.4.1 控制寄存器(CAN_CTL)
**地址偏移**：0x00
**复位值**：0x0001 0002
| 位 | 名称 | 访问 | 描述 | 复位值 |
| ---- | ---- | ---- | ---- | ------ |
| 0 | IWMOD | rw | 初始化工作模式：0-禁用初始化工作模式；1-使能初始化工作模式 | 1 |
| 1 | SLPWMOD | rw | 睡眠工作模式：0-正常工作模式；1-睡眠工作模式 | 0 |
| 2 | TFO | rw | 发送FIFO溢出：0-未溢出；1-溢出 | 0 |
| 3 | RFOD | rw | 禁用接收FIFO满时覆盖：0-使能接收FIFO满时覆盖；1-禁用接收FIFO满时覆盖 | 0 |
| 4 | ARD | rw | 自动重发禁止：0-使能自动重发；1-禁用自动重发 | 0 |
| 5 | AWU | rw | 自动唤醒：0-软件手动唤醒；1-硬件自动唤醒 | 0 |
| 6 | ABOR | rw | 自动离线恢复：1-硬件自动从离线状态恢复 | 0 |
| 7 | TTC | rw | 时间触发通信：0-禁用；1-使能 | 0 |
| 8-14 | 保留 | - | 必须保持复位值 | - |
| 15 | SWRST | rs | 软件复位：0-正常操作；1-复位CAN并进入睡眠工作模式，该位自动清0 | 0 |
| 16 | DFZ | rw | 调试冻结：当DBG_CTL中CANx_HOLD置位时生效，0-Debug时CAN收发正常；1-Debug时CAN收发停止 | 0 |
| 17-31 | 保留 | - | 必须保持复位值 | - |
