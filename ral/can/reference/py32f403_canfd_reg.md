# PY32F403 CANFD 寄存器说明
所有寄存器按**硬件偏移地址升序**排列，寄存器位段描述遵循**越低比特位越靠上**的规则，同时标注**位宽、读写属性(R/W)、复位值、功能描述**，格式统一且贴合芯片手册定义。

## 24.4.1 节点配置寄存器(CANFD_TSNCR)
**地址偏移**：0x00
**复位值**：0x02010801
**位宽**：32位

| 位段       | 名称      | 访问 | 复位值 | 功能描述                                                                 |
| ---------- | --------- | ---- | ------ | ------------------------------------------------------------------------ |
| 15:0       | VERSION   | r    | 0x0801 | CAN-CTRL版本，VER_1保存主要版本，VER_0保存次要版本                        |
| 16         | CES       | rw   | 1      | CAN错误信号：0-禁用错误信号；1-启用错误信号（符合ISO 11898-1:2015定义）|
| 17         | ROP       | rw   | 0      | 限制操作：0-不使能；1-使能（传输活动时不可更改，使能后无法开始传输）|
| 18         | Res       | -    | -      | 保留位，无功能                                                           |
| 24         | Res       | -    | -      | 保留位，无功能                                                           |
| 25         | Res       | -    | -      | 保留位，无功能                                                           |
| 31:26      | Res       | -    | -      | 保留位，无功能                                                           |
