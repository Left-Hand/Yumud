# GD32A50x CANFD 寄存器说明
**CAN0基地址**：0x4001A000
**CAN1基地址**：0x4001B000
所有寄存器按**硬件偏移地址升序**排列，位段描述遵循**越低比特位越靠上**规则，标注**位宽、读写属性、复位值、功能描述**，部分寄存器有配置约束（如仅暂停模式可配）已同步说明。

## 23.5.1 控制寄存器0(CAN_CTL0)
**地址偏移**：0x00
**复位值**：0x5900000F
**配置约束**：除位30/28/25/19外，其余位仅暂停模式可配；除位31/27/24/20外，其余位可被SWRST位复位；仅按32位字访问。

| 10         | 保留      | -    | -      | 必须保持复位值                                                           |

| 12         | MST       | rw   | -      | 邮箱中止发送：0-禁用/1-启用                                               |
| 13         | LAPRIOEN  | rw   | -      | 本地仲裁优先级使能：0-禁用/1-启用                                         |
| 14         | PNMOD     | rw   | -      | 虚拟联网模式选择：0-不选择/1-选择                                         |

| 19         | PNEN      | rw   | -      | 虚拟联网模式使能：0-禁用/1-启用（仅暂停模式可配）|
